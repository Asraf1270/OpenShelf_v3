# 📦 OpenShelf v2.6.0 Release Notes — Smart Recommendations & UI Refinements
**Release Date:** April 24, 2026

OpenShelf v2.6.0 focuses on making book discovery smarter and interface interactions smoother. By introducing intelligent search recommendations, users can now find relevant books even when exact matches are scarce. We've also polished the UI and accelerated catalog browsing with expanded infinite scroll batches.

---

## ✨ Key Features & Improvements

### 🧠 Intelligent "Related Books" System
- **Contextual Suggestions:** When search queries return few or no exact results, the system now automatically recommends related books.
- **Multi-Layered Algorithm:** The recommendation engine analyzes category matching, owner clustering, and publication grouping to present the most relevant alternatives.
- **Seamless Integration:** Related books are seamlessly merged into the search results with a discrete "Related" tag to keep the experience intuitive.

### 📱 Mobile-First UI Refinements
- **Two-Column Mobile Grid:** The "Related Books" section on individual book pages (`/book/index.php`) has been redesigned to use a responsive two-column grid on mobile devices, matching the layout of the main catalog.
- **Sleeker Book Listing Layout:** Removed the vertical gap between the main header and the sticky search bar in `/books/index.php`, creating a cleaner and more unified aesthetic as users scroll.

### ⚡ Performance & UX
- **Expanded Pagination Capacity:** Increased the default infinite scroll batch limit from 12 to 25 books. This significantly reduces the number of network requests required to browse large catalogs and provides a smoother uninterrupted scrolling experience.

---

## 🛠️ Technical Changes

- **Modified `api/books.php` & `books/index.php`:** Adjusted the fallback and query parameters to support the new `limit = 25` standard for pagination.
- **Updated `books/index.php` CSS:** Refined `.books-main` padding to eliminate the unintended top margin spacing below the header.
- **Enhanced Search Logic:** Implemented `getRelatedBooksForSearch()` helper logic to intelligently append suggestions to sparse result sets.
